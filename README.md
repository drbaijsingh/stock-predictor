# 🇮🇳 Intraday Stock Predictor

## Features
- Real-time stock analysis for Indian markets
- VWAP (Volume Weighted Average Price)
- 12+ technical indicators
- ML-powered predictions
- BUY/WAIT/AVOID signals

## How to Use
1. Search for any Indian stock
2. Select timeframe (15m, 30m, 1h, 1d)
3. Click 'Predict'
4. Review indicators and verdict

## Technologies
- Streamlit
- Python 3.11+
- yFinance
- Scikit-learn
- Plotly
